# QR для Ирины

## Публичная ссылка GitHub Pages

Основной QR-код для отправки Ирине:

```txt
qr-irina-github-pages.svg
```

Он открывает приложение по адресу:

```txt
https://metometsales-ui.github.io/beg-k-mechte/
```

Этот вариант работает и через мобильный интернет, и через Wi-Fi.

## Локальная домашняя Wi-Fi сеть

QR-код находится здесь:

```txt
qr-irina-wifi.svg
```

Он открывает приложение по адресу:

```txt
http://192.168.0.18:4173/
```

## Важно

1. Компьютер с приложением должен быть включен.
2. Сервер приложения должен быть запущен:

```powershell
.\start-server.cmd
```

3. iPhone Ирины должен быть подключен к той же домашней Wi-Fi сети.
4. Если IP-адрес компьютера изменится, QR-код нужно будет пересоздать.
