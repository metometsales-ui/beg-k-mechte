# Бег к мечте

Персональное веб-приложение для бегового дневника Ирины.

## Запуск

Откройте `index.html` напрямую в браузере или запустите локальный сервер:

```powershell
.\start-server.cmd
```

После запуска страница будет доступна по адресу:

```txt
http://127.0.0.1:4173/
```

Карты Leaflet и тайлы OpenStreetMap загружаются из интернета.
